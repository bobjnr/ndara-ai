# Ndara.AI — AI-Powered Business Management App

A full-featured mobile app built for SMEs to manage their entire business from one place.

## 📱 Demo
[Watch Demo Video](YOUR_YOUTUBE_LINK_HERE)

## ✨ Features
- Inventory, sales & invoicing management
- Real-time customer messaging
- Push notifications (low stock, invoices, orders)
- AI assistant with smart business recommendations
- Analytics dashboard — revenue, leads, orders
- Cross-platform: Android & iOS

## 🛠 Tech Stack
- React Native, Expo, TypeScript
- Firebase, REST APIs
- Expo EAS (build & release)
- TestFlight (iOS) + Android distribution
